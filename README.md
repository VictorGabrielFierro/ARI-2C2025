### Instrucciones para levantar el proyecto localmente:
1. Tener instalado PostgreSQL y Node
2. Ejecutar los scripts sql guardados en la carpeta "recursos" en el orden en el cual están numerados (tener en cuenta de ejecutar los scripts a partir del 2 en la base de datos aida_db recién creadas)
3. En la carpeta raíz ejecutar `npm install` para instalar las dependencias del proyecto
4. Luego ejecutar `npm run dev` para compilar y levantar el proyecto
5. Acceder con el link http://localhost:3000/
